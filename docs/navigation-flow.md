## Условия переходов

| Действие | Откуда | Куда | Условие |
|----------|--------|------|---------|
| SSO Login | Login | Catalog | Успешная авторизация |
| Logout | Любой | Login | Клик logout в sidebar |
| Открыть чат | Catalog (AgentCard) | Chat | POST /api/sessions → новая сессия |
| Клик по чату | My Chats (строка) | Chat | GET /api/sessions/{id} |
| Новый чат | My Chats (кнопка) | Catalog | Переход на выбор агента |
| Новый пайплайн | Pipelines (кнопка) | Pipeline Editor | Пустая форма |
| Редактировать | Pipelines (карточка) | Pipeline Editor | GET /api/pipelines/{id} |
| Запустить | Pipelines (карточка) | Pipeline Run | POST /api/pipelines/{id}/run + WS |
| Создать агента | Admin (кнопка) | Agent Editor | Пустая форма |
| Редактировать | Admin (строка) | Agent Editor | GET /api/agents/{id} |
| Tool confirmation | Chat (write-op) | Modal overlay | Агент вызывает write-tool |
| Подтвердить tool | Modal | Chat | Выполнить tool, вернуть результат |
| Отклонить tool | Modal | Chat | Вернуть отказ агенту |

## WebSocket события

- **Chat**: `ws://api/ws/chat/{session_id}` — stream, tool_use, tool_result, actions, done, error
- **Pipeline Run**: `ws://api/ws/pipeline/{run_id}` — step_start, step_progress, step_done, step_error, log, done
